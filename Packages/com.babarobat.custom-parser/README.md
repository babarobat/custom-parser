# Custom Parser (Unity UPM)

Precompiled **CustomParser.dll** only — no editable C# sources in this package folder.

- Full documentation, samples, and source: [github.com/babarobat/custom-parser](https://github.com/babarobat/custom-parser)
- Install via Git URL (`#v1.0.3` or newer); see the root [README](https://github.com/babarobat/custom-parser#unity-upm-via-git) for `manifest.json` entry.
- This package includes committed Unity `.meta` files (required for immutable UPM folders). Do not use tag **v1.0.0** — it shipped without metas.

In JetBrains Rider, you can navigate into the DLL via decompiled view for reading; the package itself is not meant for in-project editing.
